About This Project

This website was created for the Web 3D Applications module.

All models, code, and assets were created by Faiz Syed

Technologies used include: HTML5, CSS3, JavaScript, Three.js, GLTFLoader, and Blender.

About

For my website, I decided to create a website that showcases the process of travelling for people that never travelled before. It is to help them understand and visuallise what the process might look like to cool the nerves of first time fliers.

3D Models

All models were made by myself using Blender. I made 5 different models, a suitcase, a boarding pass and 3 versions of crisps. Below is a screenshot of how I developed my suitcase model. There is an animation that pulls the handle up and down allowing you to do both.

For the process of making 3 switchable models for the crisps, I had to get 3 different versions of textures for the crisps. I added them using the UV Editor.

Design

For the website, I used bootstrap to contain the models. That way, I am able to create a professional webspace for viewers to browse the website with ease. In addition, I also used bootstrap to create the navbar allowing for more a professional feel when navigating through the website. I used three.js to add light to my scene and create a spot light that will allow users to see the can in high definition. Using GLTFLoader and Java script, I was able to integrate my model for users to enjoy as well as adding interactivity to the models.

Integration

I was able to integrate a range of models into my website as well as provide a range of images, texts and animations that will help the user understand what is going on in the website.

Testing

During Lab 11, I was able to test my websites features with my peers to see how well my models worked and what was missing. This process helped me as I was able to find a lot of bugs in my code and found features I could have added.
